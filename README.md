#  Movie Sentiment Analysis

A machine learning project that analyzes movie reviews and predicts whether the sentiment is **positive or negative** using Natural Language Processing (NLP).

This project demonstrates text preprocessing, EDA, feature extraction, and classification using traditional ML techniques.

---

##  Features

- Sentiment classification of movie reviews
- NLP preprocessing pipeline
- TF-IDF based feature extraction
- Lightweight and fast inference
- Easy-to-run local setup
